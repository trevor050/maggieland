# music-cleanup

Windows-first MP3 cleanup tool for massive messy libraries.

- Identifies tracks from audio fingerprinting (AcoustID + MusicBrainz)
- Writes core tags (artist/title/album/year/track)
- Renames matched files to `Title - Artist.mp3`
- Splits output into `songs/` and `non-songs/`

See `README_QUICKSTART.md` for non-technical setup instructions.
